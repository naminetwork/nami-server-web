# 🍺 Brewery 使い方ガイド

Breweryは、マインクラフトで**リアルなお酒の醸造**ができるプラグインです。
大釜で材料を煮込み、蒸留し、樽で熟成させることで、様々な効果を持つお酒を作れます。

## 1. 醸造の基本

### 📌 醸造の3ステップ

#### ステップ1: 大釜で調理
1. 大釜に水を入れる
2. 下に火（炎ブロック、マグマ、焚き火など）を設置
3. 材料を右クリックで投入
4. 指定時間煮込む（粒子の色が変化）
5. ガラス瓶で右クリックして汲み取る

#### ステップ2: 蒸留（レシピによる）
1. 醸造台にベースポーション（大釜で作ったもの）を設置
2. グロウストーンダストを燃料として使用
3. 指定回数蒸留する

#### ステップ3: 熟成（レシピによる）
1. 樽を作成（看板に「Barrel」と書く）
2. 樽を右クリックして開く
3. お酒を入れる
4. 指定時間待つ

> **重要**: 1年 = 現実時間20分です

### 🪵 樽の作り方

#### 小樽（6スロット）
階段ブロックに「Barrel」と書いた看板を設置

#### 大樽（27スロット）
5×5×1の階段ブロックを配置し、どこかに「Barrel」と書いた看板を設置

### 🌳 樽の木材の種類

| ID | 木材 |
|:---:|:---|
| 0 | 任意の木材 |
| 1 | 白樺 |
| 2 | オーク（ジャングルウッド） |
| 3 | ジャングル |
| 4 | スプルース |
| 5 | アカシア |
| 6 | ダークオーク |
| 7 | 真紅 |
| 8 | 歪んだ |
| 9 | マングローブ |
| 10 | 桜 |
| 11 | 竹 |
| 12 | 銅の切り込み |
| 13 | ペールオーク |

> バニラマインクラフトの樽はオーク扱いです

### ⭐ 品質について

各工程の正確さで品質が3段階に分かれます：
- **低品質（+）**: 効果が弱い、副作用が強い
- **中品質（++）**: 標準的な効果
- **高品質（+++）**: 効果が強い、副作用が弱い

## 7. 酔いについて

### 🥛 酔い覚ましの方法

| 方法 | 酔い減少量 | 入手難易度 | 備考 |
|:---|:---:|:---:|:---|
| **パン** | -4 | ★☆☆☆☆ | 小麦3個でクラフト、簡単 |
| **ミルクバケツ** | -2 | ★☆☆☆☆ | **全ポーション効果も消える** |
| **アイスコーヒー** | -8 | ★★☆☆☆ | 最強、追加効果あり |
| **コーヒー** | -6 | ★☆☆☆☆ | 簡単、追加効果あり |

### ⚠️ 酔いの影響

酔いレベルが上がると：
- 画面が歪む
- チャットが文字化けする（酔っ払いの喋り方になる）
- 移動がふらつく
- 超高レベル時はログイン拒否される
- 100%到達でサーバーからキック（気絶）

### 💡 PvPでの酔い対策

1. **アイスコーヒーを大量生産**して常備する
2. **パンを最低5個**持ち歩く
3. **低〜中アルコールのお酒**を組み合わせる
4. 強いお酒を飲んだ直後に酔い覚ましを飲む
5. **ミルクバケツは最終手段**（全効果が消えるので注意）
