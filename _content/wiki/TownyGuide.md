# 🏰 Towny プレイヤー＆市長向けガイド

このガイドは、Minecraftサーバープラグイン「Towny」の基本コマンドと使い方をまとめたものです。
土地の保護、町の作成、国への参加などが可能です。

## 1. 基本・共通コマンド
誰でも使える、情報の確認やマップ表示に関するコマンドです。

| コマンド | 説明 |
| :--- | :--- |
| `/towny map` | **最重要。** チャット欄に近隣のマップを表示します。<br>`+`は自分の位置、`-`は土地、`$`は販売中の土地などを表します。 |
| `/towny prices` | 町の作成費用、維持費、土地の請求費用などの価格一覧を表示します。 |
| `/towny top land` | 土地を多く持っている町のランキングを表示します（住民数や所持金ランキングなども可）。 |
| `/res` | 自分のステータス（所属、所持金、ランクなど）を表示します。 |
| `/res <プレイヤー名>` | 他のプレイヤーのステータスを表示します。 |
| `/res friend add <名前>` | プレイヤーをフレンドに追加します（土地の権限設定などで使います）。 |
| `/res spawn` | 自分の寝ているベッド（リスポーン地点）へ移動します。 |
| `/res set mode plotborder` | **便利。** 土地の境界線をパーティクルで可視化します。歩くと表示されます。 |

## 2. 町（Town）のコマンド
町に所属することで、土地の保護や建築が可能になります。

### 一般住民向け
町に参加している人が使えるコマンドです。

| コマンド | 説明 |
| :--- | :--- |
| `/t` | 自分の所属している町の情報を表示します。 |
| `/t here` | 今立っている場所の、町の情報を表示します。 |
| `/t list` | サーバーにあるすべての町をリスト表示します。 |
| `/t online` | 同じ町のオンラインメンバーを表示します。 |
| `/t spawn` | 町のスポーン地点（広場）へテレポートします。 |
| `/t join <町名>` | 招待されている町に参加します。 |
| `/t leave` | 現在所属している町から脱退します。 |
| `/t deposit <金額>` | 町の銀行にお金を寄付します。 |

### 市長・管理職向け（町の運営）
ここからは、**町を作成する人（市長）** や、市長から権限をもらった助役などが使うコマンドです。

#### 🏠 町の作成と削除
| コマンド | 説明 |
| :--- | :--- |
| `/t new <町名>` | 新しい町を作成します。（費用がかかります） |
| `/t delete` | 町を削除（解散）します。 |

#### 🗺️ 領土（土地）の拡大
Townyは1チャンク（16x16ブロック）ごとに土地を管理します。

| コマンド | 説明 |
| :--- | :--- |
| `/t claim` | 立っている場所（チャンク）を町の領土として購入・請求します。<br>※既存の領土に隣接している必要があります。 |
| `/t claim outpost` | 飛び地（本拠地から離れた場所）を領土として請求します。（費用が高め） |
| `/t unclaim` | 立っている場所の領土を手放します。 |
| `/t unclaim all` | 町のすべての領土を放棄します（町自体は残ります）。 |

#### 👥 住民の管理・町全体トラスト
| コマンド | 説明 |
| :--- | :--- |
| `/t invite <名前>` | プレイヤーを町に招待します。 |
| `/t kick <名前>` | プレイヤーを町から追放します。 |
| `/t rank add <名前> <ランク>` | 住民にランク（役職）を付与します。ランクの詳細は下表を参照。 |
| `/t trust add <名前>` | **強力な権限です。** 指定したプレイヤーに、**町全体のすべての土地での建築・操作権限**を与えます。<br>※副市長（Assistant）にするほどではないが、町の整備を頼みたい場合などに使います。 |
| `/t trust remove <名前>` | 町全体のトラスト権限を剥奪します。 |
| `/t outlaw add <名前>` | 特定のプレイヤーを**指名手配（出入り禁止）**にします。 |

#### ⚙️ 町の設定（税金・フラグ）
| コマンド | 説明 |
| :--- | :--- |
| `/t set board <メッセージ>` | 町の掲示板（ログイン時に表示されるメッセージ）を設定します。 |
| `/t set mayor <名前>` | 市長の座を他の住民に譲ります。 |
| `/t set homeblock` | 町のホーム（中心地）を今立っているチャンクに変更します。 |
| `/t set spawn` | 町のスポーン地点（テレポート先）を今立っている場所に設定します。※Homeblock内である必要があります。 |
| `/t set taxes <金額>` | 住民から毎日徴収する税金を設定します。 |
| `/t toggle public` | **重要。** 町のスポーン地点を公開するか設定します。`off`にすると部外者は`/t spawn`で来れなくなります。 |
| `/t toggle explosion` | 町内での爆発のON/OFFを切り替えます。 |
| `/t toggle fire` | 町内での火の延焼のON/OFFを切り替えます。 |
| `/t toggle mobs` | 町内での敵対モブのスポーンのON/OFFを切り替えます。 |

### 役職・ランク（Rank）の説明
`/t rank add <名前> <ランク>` で付与できる主な役職一覧です。

| ランク名 | 意味 | 権限・役割 |
| :--- | :--- | :--- |
| **Assistant** | 助役 / 副市長 | **ほぼすべての権限を持ちます。**<br>土地の請求、住民の招待・追放、設定変更などが可能。<br>※町の削除や市長の交代だけはできません。信頼できる人にのみ与えてください。 |
| **Helper** | ヘルパー | **町の補佐役です。**<br>住民の招待や、町有地での建築許可などを持つ場合があります。<br>（Assistantほど強力ではありません） |
| **Sheriff** | 保安官 | **警察的な役割です。**<br>迷惑プレイヤーをOutlaw（指名手配）にしたり、牢屋（Jail）に入れたりする権限を持ちます。 |
| **VIP** | VIP | **名誉市民です。**<br>特別な権限はありませんが、町で設定されている**税金が免除**されます。 |
| **Donator** | 寄付者 | VIPと同様に、**税金が免除**されるランクとして使われることが多いです。 |

## 3. 土地（Plot）の管理
町の中にある個別の土地（プロット）の設定です。
市長が町全体を管理するほか、**土地を割り当てられた住民**が自分の土地を管理する際にも使います。

### トラスト（権限の共有）
**「自分の家の土地だけ、友達と一緒に建築したい」** という場合に使います。

| コマンド | 説明 |
| :--- | :--- |
| `/plot trust add <プレイヤー名>` | 立っている土地に、指定したプレイヤーを**トラスト（信頼）**追加します。<br>その人は建築やチェストの利用ができるようになります。 |
| `/plot trust remove <プレイヤー名>` | トラストを解除し、権限を剥奪します。 |
| `/plot trust list` | この土地でトラストされているプレイヤーの一覧を表示します。 |

### 土地の売買・譲渡・種類
| コマンド | 説明 |
| :--- | :--- |
| `/plot forsale <金額>` | 立っている土地を売りに出します。 |
| `/plot notforsale` | 土地の販売を取り消します。 |
| `/plot claim` | 販売されている土地を購入し、自分の所有地にします。 |
| `/plot unclaim` | 土地の所有権を放棄します。 |
| `/plot set embassy` | 土地を**大使館**に設定します。他の町の人でも購入できるようになります。 |
| `/plot set shop` | 土地をショップ用に設定します（マークが変わります）。 |
| `/plot set reset` | 土地の設定を初期状態に戻します。 |

### 土地の設定・権限（詳細）
自分の土地で「誰が何をしていいか」を細かく設定します（トラストより細かい設定が必要な場合）。

| コマンド | 説明 |
| :--- | :--- |
| `/plot set name <名前>` | 土地に名前を付けます（マップ等に表示されます）。 |
| `/plot toggle explosion` | その土地だけの爆発設定を切り替えます。 |
| `/plot toggle mobs` | その土地だけのモブ湧き設定を切り替えます（トラップタワーなどで重要）。 |

#### 🔒 権限（Perm）の設定
`Build`（建築）、`Destroy`（破壊）、`Switch`（チェストやドアの使用）、`ItemUse`（アイテム使用）を誰に許可するか設定します。

**コマンド形式:**
`/plot set perm <対象> <行動> <on/off>`

*   **対象:**
    *   `resident`: 自分
    *   `friend`: フレンドリストの人
    *   `ally`: 同じ国や同盟国の人
    *   `outsider`: 上記以外（他人）
*   **行動:** `build`, `destroy`, `switch`, `itemuse`

**よく使う例:**
*   `/plot set perm friend on` : フレンドに対して全ての権限を与える。
*   `/plot set perm outsider switch on` : 通行人でもドアやスイッチを使えるようにする。
*   `/plot set perm off` : 自分以外全ての権限を剥奪する（完全プライベート）。

## 4. 国（Nation）のコマンド
複数の町が集まって「国」を作ることができます。主に国王や市長が使用します。

| コマンド | 説明 |
| :--- | :--- |
| `/n` | 所属している国の情報を表示します。 |
| `/n list` | サーバー内の国リストを表示します。 |
| `/n new <国名>` | 新しい国を作成します（市長のみ実行可。費用がかかります）。 |
| `/n invite <町名>` | 他の町を国に招待します。 |
| `/n join <国名>` | 招待された国に加盟します。 |
| `/n leave` | 国から離脱します。 |
| `/n deposit <金額>` | 国の銀行にお金を寄付します。 |
| `/n spawn` | 国のスポーン地点（首都）へ移動します。 |
| `/n ally add <国名>` | 他の国を**同盟国**に追加します。 |
| `/n rank add <名前> <ランク>` | 国民にランク（Chancellorなど）を付与します。 |

## 5. チャット機能
Townyには専用のチャットチャンネルがあります。

| コマンド | 説明 |
| :--- | :--- |
| `/tc` | **町チャット (Town Chat)**<br>同じ町の住民にだけ見えるチャットモードに切り替えます。 |
| `/nc` | **国チャット (Nation Chat)**<br>同じ国のメンバーにだけ見えるチャットモードに切り替えます。 |
| `/lc` | **ローカルチャット**<br>近くにいる人にだけ見えるチャットです。 |
| `/g` | **グローバルチャット**<br>サーバー全体に聞こえる通常のチャットに戻します。 |
| `/res set mode map` | 移動するごとにマップをチャット欄に自動表示するモードにします（解除はもう一度入力）。 |

---

### 💡 ヒント
*   コマンドがわからないときは、単に `/towny` や `/t` と打つとヘルプが表示されることが多いです。
*   町の維持には「維持費（Upkeep）」がかかります。銀行残高（`/t`で確認）が維持費を下回ると町が消滅する可能性があるため、こまめに `/t deposit` しましょう。
