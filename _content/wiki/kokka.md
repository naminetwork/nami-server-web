# 🏰 国家鯖機能 総合ガイド

このガイドでは、当サーバーで国家運営やサバイバルを楽しむために必要な主要コマンドをまとめています。

---

## 🏗️ Towny（土地保護・国家）
町を作り、国を発展させるための根幹システムです。

| コマンド | 説明 |
| :--- | :--- |
| `/t new <町名>` | 今立っている場所を中心に新しい町を作成します。 |
| `/t deposit <金額>` | 町の銀行にお金を寄付します（維持費に必要）。 |
| `/t claim` | 今立っているチャンクを町の領土として保護します。 |
| `/t spawn` | 所属している町のスポーン地点へ移動します。 |
| `/n new <国名>` | 自分の町を首都として新しい国を建国します。 |

---

## 💎 DiamondBank（銀行）
ダイヤモンドを電子データとして安全に預けられるシステムです。

| コマンド | 説明 |
| :--- | :--- |
| `/deposit <枚数>` | 手持ちのダイヤモンドを銀行に預けます。 |
| `/withdraw <枚数>` | 銀行からダイヤモンドをアイテムとして引き出します。 |
| `/bal` | 自分の銀行残高を確認します。 |
| `/bank top` | サーバー内の富豪ランキングを表示します。 |

---

## 🧪 BottleEXP（経験値ボトル化）
経験値を「エンチャントの瓶」にして保存・取引できます。

| コマンド | 説明 |
| :--- | :--- |
| `/bottle store <数値>` | 指定した経験値量を「エンチャントの瓶」に変換します。 |
| `/bottle store all` | 所持している全ての経験値をボトル化します。 |

---

## 🌀 RTP（ランダムワープ）
資源集めや未開の地の探索に便利な、ランダムテレポート機能です。

| コマンド | 説明 |
| :--- | :--- |
| `/rtp` | 世界のランダムな地点へテレポートします。 |

---

## ⚔️ PvP Toggle（戦闘設定）
不慮の事故を防ぐため、プレイヤー同士の戦闘許可を切り替えられます。

| コマンド | 説明 |
| :--- | :--- |
| `/pvp on` | PvPを有効にします。 |
| `/pvp off` | PvPを無効にします（平和モード）。 |
| `/pvp status` | 現在のPvP設定を確認します。 |

---

### 💡 国家鯖で生き残るためのヒント
* **維持費の確認:** Townyの町や国を維持するには毎日維持費がかかります。`/t` で残高を確認し、`/t deposit` で余裕を持って入金しておきましょう。
* **資産の分散:** 持ち物がいっぱいになったらダイヤは銀行へ、経験値はボトルにしてチェストへ預けるのが賢明です。
* **平和な建築のために:** 建築に集中したいときは `/pvp off` に設定しておくことで、他のプレイヤーとのトラブルを未然に防げます。
