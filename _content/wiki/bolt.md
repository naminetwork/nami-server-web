# 🏰 鍵かけ・ブロック保護（Bolt）使い方ガイド

このガイドでは、チェスト、ドア、かまどなどのブロックを個別に保護できるプラグイン「**Bolt**」の操作方法を解説します。
土地保護（PlayerGuard）とは別に、中身やアクセス権を細かく管理したい場合に非常に便利です。

> **基本機能：** チェストやドアを設置すると、自動的にあなたの「プライベート（自分専用）」として保護されます。

## 1. 🔒 保護の基本設定
設置したブロックを手動でロック、または解除するコマンドです。

| コマンド | 説明 |
| :--- | :--- |
| `/bolt lock` | ブロックを左クリックして、自分専用に保護します。 |
| `/bolt unlock` | ブロックを左クリックして、保護を完全に解除します。 |

---

## 2. 👥 権限の変更・共有（modify）
友達と一緒にチェストを使いたい場合などは、このコマンドで「権限」を編集します。

| コマンド | 説明 |
| :--- | :--- |
| `/bolt modify <プレイヤー名> <権限> <true/false>` | ブロックを左クリックして、特定のプレイヤーなどの権限を書き換えます。 |

**よく使う例：**
* **友達を追加する：** `/bolt modify <プレイヤー名> player.full true`
  （これで相手もチェストを開けたり編集したりできるようになります）
* **権限を消す：** `/bolt modify <プレイヤー名> player.full false`

---

## 3. ⚙️ 保護タイプの変更
用途に合わせて保護タイプを変更できます。コマンド入力後にブロックを左クリックしてください。

| コマンド | 保護タイプ | 説明 |
| :--- | :--- | :--- |
| `/bolt set type private` | **プライベート** | あなた（と追加したメンバー）だけが使用できます。 |
| `/bolt set type public` | **パブリック** | 誰でも使用できますが、保護（所有権）自体は残ります。 |
| `/bolt set type donation` | **ドネーション** | 誰でもアイテムを**入れる**ことはできますが、取り出しは所有者のみです。 |

---

## 4. 📋 情報の確認と所有権の譲渡
「誰がこのチェストの持ち主か」や、権限の受け渡しに関するコマンドです。

| コマンド | 説明 |
| :--- | :--- |
| `/bolt info` | ブロックを左クリックすると、所有者や設定されている権限一覧を表示します。 |
| `/bolt transfer <プレイヤー名>` | ブロックを左クリックして、その保護の所有権を相手に譲ります。 |

---

## 5. ⚡ 便利な一括操作モード
たくさんのチェストを一度に処理したい時に便利な「永続モード」です。

| コマンド | 説明 |
| :--- | :--- |
| `/bolt mode persist` | **超便利。** このモードをONにすると、一度コマンドを打つだけで、複数のブロックを連続でクリックして同じ操作（保護や編集）ができます。 |

---

### 💡 ヒント
* `modify` コマンドは少し複雑ですが、基本的には「誰に」「どの権限を」「有効(true)か無効(false)か」を指定する流れになります。

大切なアイテムはしっかりと Bolt で保護しておきましょう！
