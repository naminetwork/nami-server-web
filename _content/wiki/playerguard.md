# 🏰 土地保護機能（PlayerGuard）使い方ガイド

サバイバルで遊んでいて、「せっかく建てた建物が荒らされないか心配…」と思ったことはありませんか？
当サーバーでは土地保護プラグイン（**PlayerGuard**）を導入しており、誰でも簡単に自分の建築物や土地を守ることができます。

> **注意：** 国家サーバーではご利用いただけません。

## 1. 🚩 土地の保護方法
金の斧を使って、直感的に範囲を指定して保護できます。

| 手順 | 操作内容 |
| :--- | :--- |
| **1. 範囲選択** | **金の斧**を持ち、保護したい範囲の対角線上の2点を指定します。<br>・1点目：ブロックを**右クリック**（始点）<br>・2点目：別の角のブロックを**右クリック**（終点） |
| **2. 保護確定** | 範囲を選択した状態で `/claim` を実行します。 |

---

## 2. 🔓 保護の解除方法
不要になった領地を削除する手順です。

| コマンド | 説明 |
| :--- | :--- |
| `/disclaim` | **注意。** 保護地域の中に立って実行すると、その領地を削除します。<br>※一度解除すると設定は元に戻せません。 |

---

## 3. ⚙️ 権限設定（フラグ）
デフォルトでは「ブロック破壊・設置は禁止」「チェスト・スイッチ類の使用は許可」されています。

| コマンド | 説明 |
| :--- | :--- |
| `/flags` | 保護地域に立って実行します。GUIやメッセージを通じて、PvPの可否や侵入制限、より詳細な権限をカスタマイズできます。 |

---

## 4. 👥 メンバー管理
自分以外のプレイヤーと一緒に作業したい場合に使用します。

| コマンド | 説明 |
| :--- | :--- |
| `/playerguard add <名前>` | 指定したプレイヤーを領地のメンバーに追加します。建築や破壊が可能になります。 |
| `/playerguard remove <名前>` | 領地のメンバーから削除し、権限を剥奪します。 |

---

## 5. 📋 領地の情報確認・譲渡
領地の詳細データ確認や、オーナー権の受け渡しができます。

| コマンド | 説明 |
| :--- | :--- |
| `/playerguard info` | 立っている領地の所有者、座標、メンバー一覧、サイズを表示します。 |
| `/playerguard transfer <名前>` | 立っている領地のオーナー権を別のプレイヤーに譲渡します。 |

---

## 6. 📈 保護できる範囲について
当サーバーでは、プレイヤーの活動量に応じて保護枠が拡大します。

* **プレイ時間連動：** サーバーで遊んでいる時間が長くなるほど、保護できる合計ブロック数（上限値）が増えていきます。
* **拡大：** 最初は小さな家から始め、長く遊ぶことでより大きな拠点や街を保護できるようになります。

---

## 7. 🧳 チェスト共有に関する注意
PlayerGuardでメンバーを追加しただけでは、チェストが開けない場合があります。その際は以下のチェスト専用ロックコマンドを使用してください。

| コマンド | 手順・説明 |
| :--- | :--- |
| `/lock <名前>` | コマンドを打った後、共有したいチェストを**左クリック**します。 |
| `/unlock` | コマンドを打った後、解除したい自分のチェストを**左クリック**します。 |

---

### 💡 ヒント
* `/playerguard` は `/pg` と略して入力できる場合があります。
* 自分の領地がどこまでか分からなくなった時は、金の斧を持って何もない場所を空クリックすると、境界線を確認できることがあります。
* 分からないことがあれば、チャットで他のプレイヤーや運営に気軽に聞いてみてください！

安心してサバイバルをお楽しみください！
