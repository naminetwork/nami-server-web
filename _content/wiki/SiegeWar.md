# ⚔️ 攻城戦 (SiegeWar) ガイド

当サーバーでは、プラグイン「SiegeWar」を使用した国家間の領土争奪戦が可能です。
無秩序な争いを防ぐため、**「合意制」**かつ**「スケジュール制」**のルールを採用しています。

## 1. 戦争の始め方（開戦フロー）

当サーバーの戦争は、**双方の合意**が必要です。

1.  **合意形成:** 外交（Discordやチャット）で相手国と戦争の合意を取ります。
2.  **平和解除:** 攻撃側・防衛側の双方が、コマンド `/t toggle peaceful` を実行し、平和主義モードを解除します。
    *   *※平和主義のままだとシステム的に宣戦布告できません。*
    *   *※当サーバーでは平和主義の維持費は無料、切り替え待機時間は0秒です。*
3.  **バナー準備:** 攻撃側は、羊毛と棒で**「バナー（旗）」**をクラフトしてください。
    *   *※通常のマイクラの旗です。白以外の色を使用してください（白旗は降伏用です）。*
4.  **宣戦布告 (設置):** **土曜日** に、攻撃側の王または将軍が「バナー」を持って相手の町の境界線（Wilderness）に行き、**地面に設置**します。
    *   *※設定により、町の境界線に隣接した場所でないと設置できません。*
5.  **コスト支払い:** 設置するとチャットに費用の確認メッセージが表示されます。
    *   チャット内の **[Confirm]** をクリックするか、コマンド `/sw confirm` を入力すると、国家銀行から費用が引き落とされ開戦します。

### 💰 戦争にかかる費用
戦争は「賭け試合」です。負けると資金を没収されます。
当サーバーでは首都への特別追加料金はありません。どの都市も平等に「広さと人口」で計算されます。

| 項目 | 1プロットあたりの単価 | 説明 |
| :--- | :--- | :--- |
| **手付金** | **$15.0** | サーバーに支払う手数料（返金されません）。 |
| **軍資金** | **$35.0** | 勝利すれば返金、敗北すれば敵に没収される賭け金。 |
| **合計** | **$50.0** | **開戦時に最低限必要な額。** |

> **⚠️ 注意: 人口によるコスト変動**
> 人口（Townyレベル）が高い街ほど、コストが少しずつ加算されます。
> *   レベルが1上がるごとに **+5%** の補正がかかります。
> *   最大レベル（人口28人以上）の町は **+50%** のコストになります。

## 2. 戦争スケジュール

宣戦布告から決着まで、**合計5回の戦闘（バトルセッション）**が行われます。
開戦した週の土日から始まり、**翌週の土曜日**に決着がつきます。

### 📅 開催日と時間
宣戦布告可能日: **土曜日**

**【バトルセッション時間割】**
*   **土曜日:** 10:00～11:00 / 21:00～22:00
*   **日曜日:** 10:00～11:00 / 21:00～22:00

### ⏳ 進行例（土曜朝に宣戦布告した場合）
1.  **土曜 10:00** - 第1戦 (または21:00)
2.  **土曜 21:00** - 第2戦
3.  **日曜 10:00** - 第3戦
4.  **日曜 21:00** - 第4戦
    *   *(月～金は休戦期間)*
5.  **翌週 土曜 10:00** - **最終決戦 (ここで決着)**

## 3. 戦闘ルールと仕様

戦闘中、戦争エリア（SiegeZone / バナー周辺半径200m）は以下の特殊ルールになります。

### ⚔️ PvPとデスペナルティ
*   **強制PvP:** 町の設定に関わらず、エリア内は**強制的にPvP有効**になります。
*   **アイテムロストなし:** 死亡してもインベントリのアイテムは落としません（KeepInventory有効）。
*   **装備劣化:** その代わり、装備中の防具・ツールの耐久値が **10% 減少** します。
    *   *修理費がかさむ消耗戦となります。*

### 🚫 禁止・制限事項
*   **建築制限:** バナー周辺（半径32ブロック）での建築は制限されます。
*   **ブロック設置:** 黒曜石、氷の設置はできません。
*   **バケツ:** 水、溶岩などのバケツは使用できません。
*   **爆発物:** TNT等は機能しません。
*   **テレポート:** 戦闘中は、その町の住民以外はエリア内にテレポートできません（徒歩で向かう必要があります）。

## 4. 勝敗と報酬

### 🏆 攻撃側の勝利条件
*   ポイントを **20,000点** 先取する。
*   または、5回目の戦闘終了時（翌週土曜）にポイントがプラスであること。

**【勝利報酬】**
1.  預けていた **軍資金($35.0/plot)** が全額返金されます。
2.  敵の国から **略奪金($30.0/plot)** を奪い取ります。
3.  敵の町を **「占領」** 状態にします。

### 🛡️ 防衛側の勝利条件
*   ポイントを **-20,000点** （マイナス2万点）にする。
*   または、5回目の戦闘終了時（翌週土曜）にポイントがマイナスであること。
*   攻撃側が「撤退（降伏）」した場合。

**【勝利報酬】**
1.  攻撃側が預けていた **軍資金($35.0/plot)** をすべて没収し、獲得します。
2.  町は守られます。

### 🎌 占領と反乱 (Revolt)
町が占領されると、毎日国家に対して税金を支払うことになります。
占領から3日（72時間）経過後、占領された町は**反乱**を起こすことができます。

*   **反乱の方法:** 白以外のバナーを用意し、**町の外（境界線の外）**に設置することで反乱戦争が開始されます。
*   勝利すれば独立できます。

## 5. 主要コマンド一覧

| 目的 | コマンド | 説明 |
| :--- | :--- | :--- |
| **準備** | `/t toggle peaceful` | 平和主義モードの切り替え（開戦に必須） |
| **開戦** | `/sw confirm` | バナー設置後のコスト支払いを確定する |
| **情報** | `/sw hud` | 戦況を表示するHUDのON/OFF切り替え |
| | `/n siege history` | 国家の戦争履歴を見る |
| | `/sw show <町名>` | 現在のその町の戦争状態を確認する |
| **金銭** | `/sw nation paysoldiers <金額>` | 国家の兵士（国民）に給料を分配する |
| | `/sw collect` | 勝利報酬などを受け取る |
| **決着** | `/sw surrender` | （防衛側）降伏する |
