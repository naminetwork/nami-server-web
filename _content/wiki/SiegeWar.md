# ⚔️ 攻城戦 (SiegeWar) ガイド

当サーバーでは、プラグイン「SiegeWar」を使用した国家間の領土争奪戦が可能です。
無秩序な争いを防ぐため、**「合意制」**かつ**「スケジュール制」**のルールを採用しています。

## 1. 戦争の始め方（開戦フロー）

当サーバーの戦争は、**双方の合意**が必要です。

1.  **合意形成:** 外交（Discordやチャット）で相手国と戦争の合意を取ります。
2.  **平和解除:** 攻撃側・防衛側の双方が、コマンド `/t toggle peaceful` を実行し、平和主義モードを解除します。
    *   *※平和主義のままだとシステム的に宣戦布告できません。*
    *   *※当サーバーでは平和主義の維持費は無料、切り替え待機時間は0秒です。*
3.  **バナー準備:** 攻撃側は、羊毛と棒で**「バナー（旗）」**をクラフトしてください。
    *   *※通常のマイクラの旗です。白以外の色を使用してください（白旗は降伏用です）。*
4.  **宣戦布告 (設置):** **木曜日** に、攻撃側の王または将軍が「バナー」を持って相手の町の境界線（Wilderness）に行き、**地面に設置**します。
    *   *※設定により、町の境界線に隣接した場所でないと設置できません。*
5.  **コスト支払い:** 設置するとチャットに費用の確認メッセージが表示されます。
    *   チャット内の **[Confirm]** をクリックするか、コマンド `/sw confirm` を入力すると、国家銀行から費用が引き落とされ開戦します。

### 💰 戦争にかかる費用
戦争は「賭け試合」です。負けると資金を没収されます。
当サーバーでは首都への特別追加料金はありません。どの都市も平等に「広さと人口」で計算されます。

| 項目 | 1プロットあたりの単価 | 説明 |
| :--- | :--- | :--- |
| **手付金** | **$5.0** | サーバーに支払う手数料（返金されません）。 |
| **軍資金** | **$25.0** | 勝利すれば返金、敗北すれば敵に没収される賭け金。 |
| **合計** | **$30.0** | **開戦時に最低限必要な額。** |

> **⚠️ 注意: 人口によるコスト変動**
> 人口（Townyレベル）が高い街ほど、コストが少しずつ加算されます。
> *   レベルが1上がるごとに **+5%** の補正がかかります。
> *   最大レベル（人口28人以上）の町は **+50%** のコストになります。

## 2. 戦争スケジュール

宣戦布告から決着まで、**合計7回の戦闘（バトルセッション）**が行われます。
開戦した週の土日から始まり、**翌週の土曜日**に決着がつきます。

### 📅 開催日と時間
宣戦布告可能日: **木曜日**

**【バトルセッション時間割】**
*   **土曜日:** 10:00～11:00 / 15:00～16:00 / 21:00～22:00
*   **日曜日:** 10:00～11:00 / 15:00～16:00 / 21:00～22:00

### ⏳ 進行例（木曜日に宣戦布告した場合）
1.  **土曜 10:00** - 第1戦
2.  **土曜 15:00** - 第2戦
3.  **土曜 21:00** - 第3戦
4.  **日曜 10:00** - 第4戦
5.  **日曜 15:00** - 第5戦
6.  **日曜 21:00** - 第6戦
7.  **翌週 土曜 10:00** - **最終決戦 (第7戦・ここで決着)**

## 3. 軍事ランクの割り当て

戦闘に参加する前に、王や市長は軍事ランクを割り当てる必要があります。

### 🎖️ 国家の軍事ランク
*   **Private（二等兵）, Sergeant（軍曹）, Lieutenant（中尉）, Captain（大佐）, Major（少佐）, Colonel（大佐）:** バトルポイントを獲得できます。
*   **General（将軍）:** 上記に加え、国家の軍事ランク割り当て、攻城戦の開始/放棄、略奪/占領が可能です。

### 🎖️ 町の軍事ランク
*   **Guard（警備兵）:** バトルポイントを獲得できます。

**ランク割り当てコマンド:**
*   町: `/t rank add [プレイヤー名] guard` - 市長がガードランクを付与
*   国家: `/n rank add [プレイヤー名] [ランク]` - 王がランクを付与（private, sergeant等）

## 4. 戦闘ルールと仕様

戦闘中、戦争エリア（SiegeZone / バナー周辺半径300m）は以下の特殊ルールになります。

### ⚔️ PvPとデスペナルティ
*   **強制PvP:** 町の設定に関わらず、エリア内は**強制的にPvP有効**になります。
*   **アイテムロストなし:** 死亡してもインベントリのアイテムは落としません（KeepInventory有効）。
*   **装備劣化:** その代わり、装備中の防具・ツールの耐久値が **5% 減少** します。
    *   *修理費がかさむ消耗戦となります。*

### 🏴 バトルセッション中のチャット制限
*   **一般チャット、/lc、/tell は無効化:** バトルセッション中（および終了後10分間）は使用できません。
*   **/tc、/nc、/ac は使用可能:** チームメイトとの連携は可能です。

### ⚔️ 戦闘中のアクション

#### 📍 スポーン
*   各陣営に1名ずつ**バトルコマンダー**が自動指定されます（王または将軍で、攻城区域内にいるプレイヤー）。
*   コマンダーが生存し区域内にいる場合、`/sw spawn <町名>` でテレポート可能です。

#### 🎯 バナー占領
*   軍事ランクを持つ公式参加者として、攻城バナーに近づいてください。
*   **7分間** バナー付近に留まると、バナーを占領し「バナーコントロールリスト」に追加されます。
*   リストに載っているプレイヤー1人につき、毎分 **30バトルポイント** が自動的に加算されます。
*   敵からバナーコントロールを奪うと「バナーコントロール反転」となり、ポイント生成率が **3倍** になります。

#### ⚔️ 敵の撃破
*   軍事ランクを持つ敵の公式参加者を倒すと、**150バトルポイント** を獲得します。

### 🚫 禁止・制限事項
*   **建築制限:** バナー周辺（半径32ブロック）での建築は制限されます。
*   **ブロック設置:** 黒曜石、氷の設置はできません。
*   **バケツ:** 水、溶岩などのバケツは使用できません。
*   **爆発物:** TNT等は機能しません。

## 5. 勝敗と報酬

### 🏆 攻撃側の勝利条件
*   7回の戦闘終了時に **Siege Balance（攻城バランス）が0より大きい** こと。
    *   *※攻城バランスは-20,000～+20,000の間で推移します。*

**【勝利報酬】**
1.  預けていた **軍資金($20/plot)** が全額返金されます。
2.  敵の町を **「略奪」** および/または **「占領」** できます。

### 🛡️ 防衛側の勝利条件
*   7回の戦闘終了時に **Siege Balanceが1未満** であること。
*   または、攻撃側が「撤退（降伏）」した場合。

**【勝利報酬】**
1.  攻撃側が預けていた **軍資金($20/plot)** をすべて没収し、獲得します。
2.  町は守られます。

### 🏴‍☠️ 攻撃の放棄
*   攻撃側の王または将軍は、町の境界付近に**全て白のバナー**を設置することで攻撃を放棄できます。

### 🏳️ 防衛の降伏
*   防衛側の市長は、町の境界付近に**全て白のバナー**を設置することで降伏できます。

---

## 6. 略奪と占領

### 💰 略奪（Plunder）
*   勝利した王/将軍は、町の境界付近に**チェスト**を設置します。
*   この行動により、町から **$40/plot** を奪い、国家銀行に入金します。

### 🚩 占領（Invade/Capture）
*   勝利した王/将軍は、町の境界付近に**白以外のバナー**を設置します。
*   この行動により、町を占領し、勝利国の一部として追加します（首都は占領不可）。
*   占領された町は「占領状態（Occupied）」となり、元の国家の /nation および /alliance チャットには残りますが、`/n leave` では脱退できません。

---

## 7. 反乱 (Revolt)

占領された町の市長は、**反乱攻城戦**を開始することで占領から脱出できます。

### 🗽 反乱の開始方法
*   市長として、町の外（境界線の外）に**白以外のバナー**を設置します。
*   これにより即座に占領が解除され、反乱攻城戦が開始されます。

### 📋 反乱攻城戦の結果
*   **町が勝利:** 占領から解放されたままです。
*   **町が敗北:** 占領国が再占領および/または略奪できます。

---

## 8. 攻城戦後の免疫期間

### 🛡️ 攻城免疫 (Siege Immunity)
*   攻城戦後、町は **7日間** の攻城免疫を獲得します（首都は **14日間**）。
*   この期間中、町は再攻撃を受けません。

### 🗽 反乱免疫 (Revolt Immunity)
*   攻城戦後、町は **3日間** の反乱免疫を獲得します。
*   この期間中、町は反乱を起こせません。

## 9. 平和主義モード

### ☮️ 平和主義町にする
*   コマンド: `/t toggle peaceful`
*   市長として、町を平和主義にできます（首都を除く）。
*   平和主義状態は **7日後** に確定します（新しい町は **2日後**）。

**メリット:**
*   攻城戦の攻撃を受けなくなります。

**デメリット:**
*   町のホームブロックを移動できません。
*   住民は国家の軍事ランクを受け取れません。
*   近隣の国家によって**転覆（Subvert）**される可能性があります。

#### 転覆の仕組み
*   平和主義町から1,200ブロック以内の、最大の（タウンブロック数で）非攻城中・非平和主義の町が「守護町」となります。
*   守護町を所有する国家は、平和主義町の境界に白以外のバナーを設置することで、即座に転覆し占領できます。

### ☮️ 平和主義を解除する
*   コマンド: `/t toggle peaceful`
*   非平和主義状態は **7日後** に確定します（新しい町は **2日後**）。

## 10. 主要コマンド一覧

### 🛠️ 準備・設定
| コマンド | 説明 |
| :--- | :--- |
| `/t toggle peaceful` | 平和主義モードの切り替え（開戦に必須）。 |
| `/sw preference bossbar [on/off]` | 戦争時のボスバー表示をON/OFFします。 |
| `/sw preference beacon [on/off]` | 戦争時の光の柱（ビーコン）表示をON/OFFします。 |

### 🎖️ 軍事ランク
| コマンド | 説明 |
| :--- | :--- |
| `/t rank add [プレイヤー名] guard` | （市長用）町の住民にガードランクを付与します。 |
| `/n rank add [プレイヤー名] [ランク]` | （国王用）国民に軍事ランクを付与します。<br>※ランク: private, sergeant, lieutenant, captain, major, colonel |
| `/n rank add [プレイヤー名] general` | （国王用）将軍ランクを付与します。将軍は自分で他のランクを付与できます。 |

### ⚔️ 戦争・情報
| コマンド | 説明 |
| :--- | :--- |
| `/sw hud [町名]` | 戦況を表示する画面（HUD）の表示/非表示を切り替えます。 |
| `/sw spawn [町名]` | 戦争中の町へテレポートします（バトルコマンダーが生存している場合）。 |
| `/t` / `/t here` / `/t [町名]` | 町の詳細情報（攻城戦の状況を含む）を表示します。 |
| `/n [国家名]` | 国家が関与している攻城戦のリストを表示します。 |
| `/sw confirm` | 宣戦布告時のコスト支払いを確定し、戦争を開始します。 |

### 💰 金銭・報酬
| コマンド | 説明 |
| :--- | :--- |
| `/sw nation paysoldiers [金額]` | （国王用）国家の兵士（国民）に給料を一括分配します。<br>※ランクに応じて配分されます。 |
| `/sw collect` | （国民用）配分された給料や報酬を受け取ります。 |
| `/n set occupationtax [税率]` | （国王用）占領税率を設定します。 |
| `/n set occupationtax max` | （国王用）占領税をサーバー設定の最大値に設定します。 |

### 🏁 決着・その他
| コマンド | 説明 |
| :--- | :--- |
| `/sw surrender` | （防衛側の市長）降伏し、敗北を認めます。※白旗設置でも可 |
| `/sw revolt` | （占領下の市長）反乱について確認します。※バナー設置で開始 |
