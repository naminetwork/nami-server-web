# ⚔️ 攻城戦 (SiegeWar) ガイド

当サーバーでは、プラグイン「SiegeWar」を使用した国家間の領土争奪戦が可能です。
ただし、無秩序な争いを防ぐため、**「合意制」**かつ**「スケジュール制」**のルールを採用しています。

## 1. 戦争の始め方（開戦フロー）

当サーバーの戦争は、**双方の合意**が必要です。

1.  **合意形成:** 外交（Discordやチャット）で相手国と戦争の合意を取ります。
2.  **平和解除:** 攻撃側・防衛側の双方が、コマンド `/t toggle peaceful` を実行し、平和主義モードを解除します。
    *   *※平和主義のままだとシステム的に宣戦布告できません。*
3.  **宣戦布告:** **木曜日** に、攻撃側が「攻城バナー」を持って相手の町の境界線に行き、地面に設置します。
    *   コマンド: `/sw start` （バナーを持った状態で入力）
4.  **コスト支払い:** 宣戦布告と同時に、国家銀行から軍資金が引き落とされます。

### 💰 戦争にかかる費用
戦争は「賭け試合」です。負けると資金を没収されます。
当サーバーでは**首都への特別追加料金はありません。** どの都市も平等に「広さと人口」で計算されます。

| 項目 | 1プロットあたりの単価 | 説明 |
| :--- | :--- | :--- |
| **手付金** | **$15.0** | サーバーに支払う手数料（返金されません）。 |
| **軍資金** | **$35.0** | 勝利すれば返金、敗北すれば敵に没収される賭け金。 |
| **合計** | **$50.0** | **開戦時に最低限必要な額。** |

> **⚠️ 注意: 大都市補正 (人口による変動)**
> 人口（Townyレベル）が高い街ほど、コストが加算されます。
> *   人口 2人以上 (Lv2): +10%
> *   人口 6人以上 (Lv3): +20%
> *   人口 10人以上 (Lv4): +30%
> *   人口 20人以上 (Lv5): **+40%**
> *   人口 50人以上 (Lv6): **+50%**

## 2. 戦争スケジュール

宣戦布告から決着まで、**合計5回の戦闘（バトルセッション）**が行われます。
戦闘時間外は準備期間となり、攻撃できません。

| 曜日 | 時間 | 内容 |
| :--- | :--- | :--- |
| **木曜日** | 終日 | **宣戦布告が可能**な日 |
| 金曜日 | - | 準備期間 |
| **土曜日** | **10:00 - 11:00** | 第1戦 (朝の部) |
| **土曜日** | **21:00 - 22:00** | 第2戦 (夜の部) |
| **日曜日** | **10:00 - 11:00** | 第3戦 (朝の部) |
| **日曜日** | **21:00 - 22:00** | 第4戦 (夜の部) |
| 月～金 | - | 休戦・準備期間 |
| **翌週 土曜** | **10:00 - 11:00** | **最終決戦 (ここで決着)** |

## 3. 戦闘ルールと仕様

戦闘中、戦争エリア（SiegeZone / バナー周辺半径200m）は以下の特殊ルールになります。

### ⚔️ PvPとデスペナルティ
*   **強制PvP:** 町の設定に関わらず、エリア内は**強制的にPvP有効**になります。
*   **アイテムロストなし:** 死亡してもインベントリのアイテムは落としません。
*   **装備劣化:** 死亡ペナルティとして、装備中の防具・ツールの耐久値が **10% 減少** します。
    *   *修理費がかさむ消耗戦となります。*

### 🚫 禁止・制限事項
*   **建築制限:** バナー周辺（半径32ブロック）での建築は制限されます。
*   **ブロック設置:** 黒曜石、氷の設置はできません。
*   **バケツ:** 水、溶岩などのバケツは使用できません。
*   **爆発物:** TNT等は機能しません。

## 4. 勝敗と報酬

### 🏆 攻撃側の勝利条件
*   ポイントを **20,000点** 先取する。
*   または、期間終了時（翌週土曜11:00）にポイントがプラスであること。

**【勝利報酬】**
1.  預けていた **軍資金($35.0/plot)** が全額返金されます。
2.  敵の国から **略奪金($30.0/plot)** を奪い取ります。
3.  敵の町を **「占領」** 状態にします。

### 🛡️ 防衛側の勝利条件
*   ポイントを **-20,000点** （マイナス2万点）にする。
*   または、期間終了時（翌週土曜11:00）にポイントがマイナスであること。
*   攻撃側が「撤退（降伏）」した場合。

**【勝利報酬】**
1.  攻撃側が預けていた **軍資金($35.0/plot)** をすべて没収し、獲得します。
2.  町は守られます。

## 5. よくある質問 (FAQ)

**Q. 相手が平和主義を解除してくれません。**
A. 当サーバーは合意制です。解除してくれない場合、戦争はできません。粘着して強要する行為は迷惑行為として処罰対象になります。

**Q. 戦争中に町にテレポートできますか？**
A. バトルセッション中（戦闘時間中）は、その町の**住民のみ**スポーン可能です。部外者はエリア外から徒歩で向かう必要があります。

**Q. 占領されたらどうなりますか？**
A. 毎日、国家に対して「税金」を取られます。また、3日（72時間）経過後に「反乱」を起こして独立戦争を仕掛けることができます。

**Q. 第三勢力は介入できますか？**
A. 可能です。ただし、エリア内に入ると強制PvP状態になるため、どちらの勢力からも攻撃されるリスクがあります。

## 6. 主要コマンド一覧

| コマンド | 説明 |
| :--- | :--- |
| `/sw hud` | 戦況を表示するHUDのON/OFF切り替え |
| `/sw nation paysoldiers <金額>` | 国家の兵士（国民）に給料を分配する |
| `/sw collect` | 勝利報酬などを受け取る |
| `/t toggle peaceful` | 平和主義モードの切り替え（開戦に必須） |
| `/n siege history` | 国家の戦争履歴を見る |
| `/sw revolt` | 占領された町が反乱を起こす |
