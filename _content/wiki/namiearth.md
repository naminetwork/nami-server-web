# 🏰 NamiEarth機能 総合ガイド

NamiEarthでは、**金（ゴールド）を基軸通貨とした経済システム**と、**土地保護・国家運営システム**が連動しています。快適に建国・生活を楽しむために、以下の主要コマンドを活用してください。

---

## 🗳️ 投票 (Vote)
サーバーに投票することで、特典を受け取ることができます。
**※NamiEarthにログインしている状態**で、各サイトにログインし、自分のMCIDを入力して投票してください。

1. **[JMS (日本マインクラフトサーバー)](https://minecraft.jp/servers/nami.mcsv.win)** - **サイトへのログイン必要**
2. **[MinecraftServers.org](https://minecraftservers.org/server/681665)**
3. **[Mineportal](https://mineportal.jp/servers/cmg0klc4c000013jiwqt04cw4/edit)**

## 💰 GoldBank（基軸通貨・銀行）
NamiEarthにおいて、金（ゴールド）は単なる素材ではなく「通貨」です。
銀行に預けることで、維持費の支払いやプレイヤー間の経済活動が可能になります。

| コマンド | 説明 |
| :--- | :--- |
| `/deposit <枚数>` | 手持ちの金インゴットを銀行（通貨残高）に預けます。 |
| `/withdraw <枚数>` | 銀行から金インゴットをアイテムとして引き出します。 |
| `/bal` | 自分の現在の所持金（銀行残高）を確認します。 |
| `/bank top` | サーバー内の富豪ランキングを表示します。 |

## 🏗️ Towny（国家運営・土地保護）
町を作り、国を発展させるためのシステムです。作成には銀行残高が必要です。
詳細は **[Towny解説Wiki](https://nami-minecraft.vercel.app/wiki/towny)** を確認してください。

| コマンド | 説明 |
| :--- | :--- |
| `/t new <町名>` | 今立っている場所を中心に町を作成します。**（費用: 64g）** |
| `/t deposit <金額>` | 町の銀行にお金を寄付します。**（町の維持費になります）** |
| `/t claim` | 今立っているチャンクを町の領土として保護します。 |
| `/t spawn` | 所属している町のスポーン地点へ移動します。 |
| `/n new <国名>` | 自分の町を首都として新しい国を建国します。**（費用: 2048g）** |

## 🔒 Bolt（個人資産保護）
チェストやドアをロックし、盗難を防ぎます。
詳細は **[Bolt解説Wiki](https://nami-minecraft.vercel.app/wiki/bolt)** を確認してください。

| コマンド | 説明 |
| :--- | :--- |
| `/bolt lock` | 対象を自分専用にロックします（設置時に自動適用）。 |
| `/bolt unlock` | ロックを解除します。 |
| `/bolt modify +<名前>` | 指定したプレイヤーに操作権限（共有設定）を追加します。 |
| `/bolt info` | 保護の状態や権限リストを確認します。 |

## ⛏️ Cherry Drill（特殊ツール）
投票ガチャから**低確率**で入手できるレアアイテムです。3×3範囲を一度に破壊できる特殊なネザライトピッケルで、経験値でチャージして使用します。

- **入手方法**: 投票ガチャ限定（低確率）
- **特徴**: 耐久値無限・効率強化V・3×3範囲破壊
- **チャージ**: Shiftキーで経験値チャージ（20XP = 1日分）
- **⚠️ 注意**: チャージが0になると完全に消滅します

### 💡 NamiEarthで生き残るためのヒント
* **資産の守り方:** 貴重な金（ゴールド）は `/deposit`、チェストは `/bolt`、経験値は `/bottle`。この3点を徹底しましょう。
* **名称の制限:** 町名・国名に日本語を使うことはできません。必ず半角英数字で命名してください。
* **維持費の重要性:** 町の銀行残高がなくなると維持費が払えず、**そのうち町が消滅（崩壊）してしまいます。** 定期的に `/t deposit` して蓄えを確認しましょう。
* **町と国の作成費用:** 町の作成には **64g**、建国には **2048g** が銀行残高から引き落とされます。
* **⚠️ データの長期保管について:**
    > **30日以上ログインがない場合、Townyの土地データおよびBoltのチェスト保護データは自動的に削除（解放）されます。** 長期間不在にする場合はご注意ください。
