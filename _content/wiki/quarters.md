# 🏘️ Quarters 使い方ガイド

Quartersは、Townyの土地システムを拡張した**3D区画システム**です。
通常のPlotはチャンク単位の平面管理ですが、Quartersでは**立体的で自由な形の区画（アパート）**を作成・管理できます。

## 1. Quartersでできること

### 通常のPlotとの違い
- **Plot**: 16x16ブロックのチャンク単位。平面的で固定サイズ。
- **Quarter**: 自由な大きさの**立方体（3D）**を組み合わせて作成可能。

### 使用例
- マンションの各部屋を個別に区画化
- 建物の1階と2階で別の所有者を設定
- 複雑な形の店舗スペースを作成

## 2. 区画の作成（市長・助役向け）

区画を作成できるのは**市長**または**助役（Assistant）**だけです。

### 📌 基本的な作成手順

#### ステップ1: ワンドを入手
火打石（Flint）がワンドです。初回のみコマンドで無料入手可能:
```
/q wand
```

#### ステップ2: 範囲を選択
火打石を手に持ち、区画にしたい場所の**角1**と**角2**を右クリック/左クリックで選択します。
立方体の輪郭がパーティクルで表示されます。

> コマンドでも位置指定可能: `/q pos 1` と `/q pos 2`

#### ステップ3: 区画を作成
単純な立方体ならそのまま:
```
/q create
```

複雑な形にする場合は `/q selection add` で立方体を追加してから `/q create`

### 🔧 選択の編集
| コマンド | 説明 |
| :--- | :--- |
| `/q selection add` | 選択した立方体を追加（複数組み合わせ可能） |
| `/q selection remove` | 立っている立方体を選択から削除 |
| `/q selection clear` | すべての選択をクリア |
| `/q selection copy` | 選択をコピー（現在位置基準） |
| `/q selection paste` | コピーした選択を貼り付け |

### ⚠️ 作成時の注意点
以下の条件をすべて満たす必要があります:
- 区画全体が1つの町の中にある
- 区画内に荒野（未請求地）がない
- 既存の区画と重ならない
- ワールドの高度制限内である

## 3. 区画の購入・所有

### 区画を購入する
販売中の区画の中に立って:
```
/q claim
```

### 区画を売りに出す
自分が所有している区画で:
```
/q sell <金額>
```
例: `/q sell 500` → 500ゴールドで販売

### 区画を手放す
```
/q unclaim
```

### 区画の情報を見る
立っている場所の区画情報を表示:
```
/q here
```

## 4. 区画の権限管理

### 🤝 トラスト（信頼）機能
**友達と一緒に建築したい場合**に使います。トラストされた人は、所有者と同じ権限（建築・破壊・チェスト使用など）を得られます。

| コマンド | 説明 |
| :--- | :--- |
| `/q trust add <プレイヤー名>` | 指定したプレイヤーをトラストに追加 |
| `/q trust remove <プレイヤー名>` | トラストを解除 |
| `/q trust clear` | すべてのトラストを解除 |

### 🔒 詳細な権限設定
区画ごとに「誰が何をできるか」を細かく設定できます。

**コマンド形式:**
```
/q set perm <対象> <行動> <on/off>
```

**対象の種類:**
- `resident` : 自分
- `friend` : フレンドリスト（`/res friend add`）の人
- `ally` : 同盟国の人
- `outsider` : それ以外の人

**行動の種類:**
- `build` : ブロック設置
- `destroy` : ブロック破壊
- `switch` : ドア・ボタン・レバーの使用
- `itemuse` : アイテム使用

**よく使う例:**
```
/q set perm friend on
```
↑ フレンドに全権限を与える

```
/q set perm outsider switch on
```
↑ 通行人でもドア・ボタンを使えるようにする

```
/q set perm off
```
↑ 自分以外のすべての権限を剥奪（完全プライベート）

> **重要**: Quarter権限は、その下にあるPlot権限より**優先**されます。

## 5. 区画のタイプ

市長・助役は区画のタイプを変更できます。タイプによって特別な機能が使えます。

| タイプ | 説明 | コマンド |
| :--- | :--- | :--- |
| **Apartment**<br>（アパート） | デフォルトの区画タイプ。通常の住居用。 | - |
| **Inn**<br>（宿屋） | 区画内でベッドが使用可能になります。<br>ホテルや宿屋に最適。 | `/q set type inn` |
| **Station**<br>（駅） | 区画内でボートやトロッコなどの乗り物が使用可能。<br>駅や港に最適。 | `/q set type station` |

### 大使館モード
他の町のプレイヤーにも区画のタイプの恩恵を与えたい場合:
```
/q toggle embassy
```
↑ これをONにすると、町の外の人も宿屋でベッドを使えるようになります。

## 6. 便利な設定

### 🎨 区画の色を変える（Premium限定）
Premiumプレイヤーは区画の輪郭の色をカスタマイズ可能:
```
/q set colour <R> <G> <B>
```
例: `/q set colour 255 0 0` → 赤色

### 📛 区画に名前を付ける
```
/q set name <名前>
```
↑ `/q here` で表示される名前を設定

### 👁️ 輪郭の表示設定
| コマンド | 説明 |
| :--- | :--- |
| `/q toggle constantoutlines` | 区画の輪郭を常に表示するか切り替え |
| `/q toggle entryblinking` | 区画に入ったときに輪郭を点滅させるか切り替え |
| `/q set particlesize <サイズ>` | パーティクルの大きさを変更（0.1～1.0） |

### 💬 通知設定
| コマンド | 説明 |
| :--- | :--- |
| `/q toggle entrynotifications` | 区画に入ったときの通知のON/OFF |
| `/q set entrynotifications <chat/hotbar>` | 通知の表示場所を変更（チャット or ホットバー上） |

## 💡 よくある使い方

### マンションを作る
1. 市長が建物の1階部分を選択して `/q create`
2. 2階部分を選択して `/q create`（別の区画として作成）
3. 各階を `/q sell 1000` などで販売
4. 住民が `/q claim` で購入
5. 住民は `/q trust add <友達>` で共同建築

### ホテルを運営する
1. 各部屋を区画として作成
2. `/q set type inn` で宿屋タイプに設定
3. `/q toggle embassy` で大使館モードON
4. 旅人が宿泊費を払って `/q claim`
5. 部屋でベッドが使えるようになる

### ショッピングモールを作る
1. 各店舗スペースを区画として作成
2. `/q set name ショップA` などで名前を付ける
3. `/q sell 5000` で店舗を販売
4. 購入者は `/q set perm outsider switch on` で客がドアを使えるように設定
